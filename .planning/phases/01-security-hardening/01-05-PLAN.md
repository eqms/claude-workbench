can be safely merged.

key is committed. Operator instructions and smoke-test script exist. Plan 06 is blocked until
this plan has shipped at least 2 signed releases.

**Create `scripts/generate-signing-key.sh`:**

```bash
#!/usr/bin/env bash
# generate-signing-key.sh — one-time operator script to generate zipsign ed25519 keypair
# Run this ONCE. Store priv.key only in GitHub Actions Secret. Commit pub.bin only.
set -euo pipefail

SIGNING_DIR="$(cd "$(dirname "$0")/.." && pwd)/signing"
mkdir -p "$SIGNING_DIR"

if [ -f "$SIGNING_DIR/claude-workbench-pub.bin" ]; then
    echo "ERROR: $SIGNING_DIR/claude-workbench-pub.bin already exists."
    echo "Refusing to overwrite — this would invalidate all previously signed releases."
    exit 1
fi

echo "Installing zipsign..."
cargo install zipsign

echo "Generating ed25519 keypair..."
cd "$SIGNING_DIR"
zipsign gen-key claude-workbench-pub.bin priv.key

PUB_SIZE=$(wc -c < claude-workbench-pub.bin)
PRIV_SIZE=$(wc -c < priv.key)

if [ "$PUB_SIZE" -ne 32 ]; then
    echo "ERROR: pub.bin is $PUB_SIZE bytes, expected 32"
    exit 1
fi
if [ "$PRIV_SIZE" -ne 64 ]; then
    echo "ERROR: priv.key is $PRIV_SIZE bytes, expected 64"
    exit 1
fi

echo ""
echo "=== Next steps ==="
echo "1. Encode private key for GitHub Secret:"
echo "   base64 signing/priv.key"
echo ""
echo "2. Add to GitHub: Settings -> Secrets -> Actions -> New secret"
echo "   Name:  CLAUDE_WORKBENCH_SIGNING_KEY"
echo "   Value: (paste base64 output)"
echo ""
echo "3. Delete private key from disk:"
echo "   rm -f signing/priv.key"
echo ""
echo "4. Add to .gitignore:"
echo "   echo 'signing/priv.key' >> .gitignore"
echo ""
echo "5. Commit signing/claude-workbench-pub.bin (NOT priv.key):"
echo "   git add signing/claude-workbench-pub.bin .gitignore"
echo "   git commit -m '[ADD] sec: add ed25519 public key for release signing'"
echo ""
echo "Public key: $SIGNING_DIR/claude-workbench-pub.bin ($PUB_SIZE bytes)"
echo "KEEP priv.key ONLY as GitHub Secret — delete it from disk now."
```

**Create `scripts/test-signed-update.sh`:**

```bash
#!/usr/bin/env bash
# test-signed-update.sh — manual smoke test for signed release archive verification
# Usage: ./scripts/test-signed-update.sh [version]
# Example: ./scripts/test-signed-update.sh v0.90.0
set -euo pipefail

VERSION="${1:-}"
if [ -z "$VERSION" ]; then
    echo "Usage: $0 <version>"
    echo "Example: $0 v0.90.0"
    exit 1
fi

# Strip leading 'v' for download URL
VERSION_CLEAN="${VERSION#v}"
REPO="eqms/claude-workbench"

# Detect platform
case "$(uname -s)-$(uname -m)" in
    Darwin-arm64)  TARGET="aarch64-apple-darwin" ;;
    Darwin-x86_64) TARGET="x86_64-apple-darwin" ;;
    Linux-aarch64) TARGET="aarch64-unknown-linux-gnu" ;;
    Linux-x86_64)  TARGET="x86_64-unknown-linux-gnu" ;;
    *) echo "Unsupported platform: $(uname -s)-$(uname -m)"; exit 1 ;;
esac

ARCHIVE="claude-workbench-${TARGET}.tar.gz"
URL="https://github.com/${REPO}/releases/download/${VERSION}/${ARCHIVE}"
PUB_KEY="$(cd "$(dirname "$0")/.." && pwd)/signing/claude-workbench-pub.bin"

if [ ! -f "$PUB_KEY" ]; then
    echo "ERROR: Public key not found at $PUB_KEY"
    echo "Have you run scripts/generate-signing-key.sh and committed signing/claude-workbench-pub.bin?"
    exit 1
fi

TMPDIR_WORK="$(mktemp -d)"
trap 'rm -rf "$TMPDIR_WORK"' EXIT

echo "Downloading $ARCHIVE from $URL..."
curl -fsSL -o "$TMPDIR_WORK/$ARCHIVE" "$URL"

echo "Verifying signature..."
if zipsign verify tar "$TMPDIR_WORK/$ARCHIVE" "$PUB_KEY"; then
    echo "PASS: Signature valid for $ARCHIVE"
else
    echo "FAIL: Signature verification failed for $ARCHIVE"
    exit 1
fi
```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| GitHub Actions runner → signing key | Key must exist only in Secret; never in runner filesystem after step |
| git history → private key | A single accidental commit of priv.key requires key rotation |
| release archive → GZIP comment | zipsign embeds signature in GZIP comment; format must match client expectation |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-01 | Tampering | Release archive without signature | mitigate | CI signing step runs for every archive before upload; `if: always()` shred ensures key cleanup |
| T-05-02 | Information Disclosure | `priv.key` in git history | mitigate | `.gitignore` entry + explicit task step to verify not tracked; CI secret is the only authorized location |
| T-05-03 | Tampering | Signing key compromise via CI logs | mitigate | `base64 -d > /tmp` decode is not echoed; shred step cleans up immediately after signing |
</threat_model>
